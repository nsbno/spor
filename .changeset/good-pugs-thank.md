**Pagination:**
Pagination has changed and been converted to composition pattern. Please see docs for info.

Prop changes:

- `totalPages` > `count`
- `selectedPage` > `page`

New props: `pageSize`, `siblingsCount`, `defaultPage`, `onPageChange`, `onPageSizeChage`, `page`
See docs for more info
